Comment jouer

Un code secret de quatre glyphes à percer en six essais. À chaque proposition, le plateau vous répond.

Bien placéle bon glyphe, à la bonne position.
Présentle glyphe est dans le code, mais ailleurs.
Absentce glyphe n'est pas (ou plus) dans le code.

Pensé pour le confort

Un jeu calme et élégant. Pas de chrono, pas de pub, pas de bruit.

Une partie par jour

Tout le monde affronte la même grille. Revenez chaque jour et entretenez votre série.

Classement mondial

Comparez vos scores avec les autres joueurs, via Game Center et le classement en ligne.

Clair & sombre

Thèmes clair et sombre, ou suivi automatique de votre système.

Mode daltonien

Chaque case porte aussi un symbole distinct, pas seulement une couleur.

Hors-ligne

100 % jouable sans connexion. Le classement se synchronise quand vous le souhaitez.

Sans publicité

Aucune pub, aucune interruption, aucun achat pour jouer.
